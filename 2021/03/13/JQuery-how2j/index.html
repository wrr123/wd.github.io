定义

JQuery是一个javascript的框架，是对javascript的一个封装。

通过JQuery可以非常方便的操作HTML的元素。

一些常规的操作：

  • 使用JQuery隐藏和显示元素
  • 理解$(function(){})的意义
  • 通过id获取元素
  • 增加监听器

常见方法

关键字 简介
val 取值
html 获取元素内容，如果有子元素，保留标签
text 获取元素内容，如果有子元素，不包含子元素标签

CSS

关键字 简介
addClass 增加class
removeClass 删除class
toggleClass 切换class
css css函数
css函数

格式：css(property, value) 或 css({p1: v1, p2: v2, ...})

可以分别给元素添加单个样式或者多个样式。

选择器

关键字 简介(*选择器)
$(“tagName”) 元素
$(“#id”) id
$(“.className”) 类
$(“selector1 selector2”) 层级
:first :last 最先，最后
:odd :even 奇偶
:hidden :visible 可见性
[attribute] 属性
:input :button … 表单对象
:disabled :checked :selected 表单对象属性
this 当前元素

筛选器

筛选器指的是在已经通过 选择器 选中了元素后，在此基础上进一步选择。

关键字 简介
first() 第一个
last() 最后一个
eq(num) 第 num 个
parent() 父元素
parents() 祖先元素
children() 儿子
find() 后代元素
siblings() 同级

属性

关键字 简介
attr 获取
attr(属性， 值) 修改
removeAttr 删除
attr和prop的区别
  • 对于自定义属性，attr可以获取，但是prop不能获取
  • 对于选中属性
    • attr只能获取初始值，无论是否改变
    • prop能够访问变化后的值，并且以true | false的布尔型返回

效果

关键字 简介

slideUp slideDown slideToggle 向上滑动 向下滑动 滑动切换
fadeIn fadeOut fadeToggle fadeTo 淡入 淡出 淡入淡出切换 指定淡入程度
animate 自定义动画效果
callback 回调函数

Ajax

JQuery对Ajax的支持，使得其对Ajax的操作简化了很多。

关键字 简介

$.get() 使用get方式提交ajax
$.post() 使用post方式提交ajax
load() 最简单的调用ajax的方式
sirialize() 格式化form下的输入数据

有时候form下的输入内容比较多，一个一个的取比较麻烦，就可以用serialize()把输入数据格式化成字符串。
