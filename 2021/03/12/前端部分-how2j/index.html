HTML

中文问题

可以在浏览器上设置编码方式为GB2312，或者在html的最前面加上编码

元素

  • 完整的HTML

    一个完整的html文件应该至少包含html元素，body元素，以及里面的内容。

  • 不完整的元素

  • 有的特殊元素

    有的特殊元素，没有内容，即开始和结束标签直接放在一起。

    比如hr, br等等。

属性

属性是用来修饰标签的。

注释

html用<!-- -->来进行注释。

基本元素
格式 说明
<h1> 到 <h6> 分别表示不同大小的标题
<p> 段落，自带换行
<b> 或 <strong> 粗体
<i> 或 <em> 斜体
<pre> 在网页上显示代码，如java代码等
<del> 删除线效果
<ins> 下划线，其中标签也可以实现下划线效果，但是不建议使用
<img> 图像
<a> 超链接
<table>，<tr>, <td> 表格, 行， 单元格
<ul>, <ol> 无序列表， 有序列表
<div>, <span> 块级元素， 内联元素，它们通常结合CSS来进行页面布局
字体
字体元素

font常用的属性有color和size，分别表示颜色和大小。

颜色表示法
  • 颜色名，如red, blue
  • 颜色对应的16进制，如#ff0000就表示红色。
内联框架

<iframe>即内联框架，通过内联框架可以实现在网页中插入网页。

表单元素
格式 说明
<input type="text"> 文本框
<input type="password"> 密码框
<form> 表单
<input type="radio"> 单选框
<input type="checked"> 复选框
<select> <option> 下拉列表
<textarea> 文本域
<input type="button"> 普通按钮
<input type="submit"> 提交按钮
<input type="reset"> 重置按钮
<input type="image"> 使用图像作为按钮进行form的提交
<button> 按钮标签

CSS

css语法

selector {property: value},即

选择器 {属性: 值}。

css选择器

  • 元素选择器

  • id选择器

  • 类选择器

  • 更准确的选择(同时根据元素名和class来选择)

注释

以/* 开始，以*/结束。

常用的CSS属性
属性 说明
width, height 尺寸大小，元素的宽度和高度
background-*（color，image） 背景（颜色，图片）
color 文字颜色
text-align 对齐
text-decoration 文本修饰
line-height 行间距
letter-spacing 字符间距
word-spacing 单词间距
text-index 首行缩进
text-transform 大小写
white-space 空白格
font-size 尺寸
font-style 风格
font-weight 粗细
font-family 字库
cursor 鼠标样式
table-layout 表格布局
border-collapse 表格边框
border-style 边框风格
border-color 边框颜色
border-width 边框宽度
padding 内边距，指元素的内容和边框之间的距离
margin 外边距，指元素边框和元素边框之间的距离
css优先级问题
  1. !important
  2. style属性
  3. style标签
  4. 外部css文件
布局

  • 绝对定位 position: absolute;

  • 相对定位 position: relative;

  • 浮动 float:left/right;

    清楚浮动带来的影响，使用 clear.

JavaScript

定义

JavaScript用于网页与用户之间的交互，完整的JavaScript由语言基础、BOM和DOM组成。

script标签

JavaScript代码必须放在script标签中，script标签可以放在html的任何地方，一般建议放在head标签里。

调试方法
  • alert调试
  • 使用浏览器自带的调试功能
  • console.log()调试
基本数据类型
关键字 简介
undefined 声明了但未赋值
Boolean 布尔
Number 数字
String 字符串
var 动态类型
typeof 变量类型判断
null 空对象、或对象不存在
作用域
  • 全局变量
  • 局部变量
事件

事件有很多种，比如鼠标移动、鼠标点击和键盘点击等等。

错误处理

JavaScript提供了一种try catch的错误处理机制，当有错误抛出的时候，可以catch住。

对象
  • 数字 Number
  • 字符串 String
  • 数组 Array
  • 日期 Date
  • Math
  • 自定义对象
BOM

浏览器对象模型（Browser Object Model）

  • Window 窗口
  • Navigator 浏览器
  • Screen 客户端屏幕
  • History 访问历史
  • Location 浏览器地址
定时器
关键字 简介
setTimeout 只执行一次
setInterval 不停的重复执行
clearInterval 终止重复执行

不要在setInterval调用的函数中使用document.write();

JSON

JSON JavaScript对象表示法（Java Script Object Notation）是一种存储数据的方式。

对象转换

  • 将字符串转换为JSON对象

  • JSON对象转换为字符串
