什么是 FreeMarker

FreeMarker 是一种模板引擎：一种基于模板的、用来生成输出文件(任何来自于 HTML 格式的文本用来自动生成源代码)的通用工具。

它是为 Java 程序员提供的一个开发包或者说是类库。

它不是面向最终用户，而是为程序员提供的可以嵌入他们的开发产品的一款应用程序。
