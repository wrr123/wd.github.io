js中替换变量使用方法

上述的方法如果需要替换的特殊字符比较多的话，会非常麻烦，故为了解决，使用下面的两个方法

encodeURI()

一般用于编码整个URI,因为URI中的合法字符都不会被编码转换。

但是下面的一些特殊字符不会被编码：

**! @ # $& * ( ) = : / ; ? + ' **

encodeURIComponent()

把URI字符串采用UTF-8编码格式转化为escape格式的字符串。

一般对传递的参数进行转换。
